## 💻 코드 품질

### 품질 기준

모든 함수가 다음 기준을 만족하나요?

**타입 힌트**:
```python
def function_name(param: type) -> return_type:
```
- [ ] 모든 매개변수에 타입 힌트
- [ ] 모든 함수에 반환 타입 힌트

**Docstring**:
```python
"""
함수 설명

Args:
    param: 설명

Returns:
    반환값 설명
"""
```
- [ ] 모든 함수에 Docstring
- [ ] 명확한 설명
- [ ] Args와 Returns 섹션

**예외 처리**:

## 🔄 TDD 실천

### 실제로 경험한 것

- [ ] 테스트를 먼저 작성했다
- [ ] 테스트 실패를 먼저 확인했다 (🔴 Red)
- [ ] 최소 코드로 통과시켰다 (🟢 Green)
- [ ] 테스트가 통과한 후 리팩토링했다 (🔵 Refactor)
- [ ] 리팩토링 후에도 테스트가 통과하는 것을 확인했다

### TDD의 가치 이해

- [ ] "테스트 먼저"의 의미를 체험했다
- [ ] 테스트가 안전망 역할을 하는 것을 경험했다
- [ ] 자신감 있게 코드를 수정할 수 있었다
- [ ] 버그를 조기에 발견하는 것을 경험했다
