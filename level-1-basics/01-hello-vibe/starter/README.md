## 🚀 시작하기

### 1. 안티그라비티 준비

- 안티그라비티 웹 인터페이스 접속

- 새 대화 시작

### 2. 첫 번째 파일 만들기

**AI에게 요청**:
```
안녕! 나는 Python을 배우고 있어.
간단한 "Hello, World!" 프로그램을 만들어줄래?
파일명은 hello.py로 해줘.
```

### 3. 코드 복사 및 실행

1. AI가 제공한 코드를 `hello.py` 파일로 저장
2. 터미널에서 실행:
   ```bash
   python hello.py
   ```
3. 출력 확인: `Hello, World!`
