# 03-ai-assistant 실습 공간

이 폴더에서 AI와 함께 코드를 작성합니다.

## 📁 시작하기

### 파일 설명

- **prompt_practice.md**: 프롬프트 개선 연습 (과제 1)
- **todo.py**: TODO 앱 구현 파일 (과제 2, 3)
- **test_todo.py**: TODO 앱 테스트 파일

## 🚀 진행 순서

### 1. 프롬프트 개선 연습

`prompt_practice.md` 파일을 열고:
- 나쁜 프롬프트를 분석
- 좋은 프롬프트로 개선
- 왜 더 나은지 설명

### 2. TODO 앱 만들기

**단계별 접근**:

#### 단계 1: 기본 구조
```
AI에게 요청:
"Python으로 TODO 앱의 기본 구조를 만들어줘.
다음 함수들이 필요해:
- add_todo(title): TODO 추가
- list_todos(): TODO 목록 반환
- complete_todo(index): TODO 완료 처리
- delete_todo(index): TODO 삭제

데이터는 리스트로 메모리에 저장해줘."
```

#### 단계 2: 테스트 작성
```
AI에게 요청:
"이 TODO 앱의 모든 함수에 대한 pytest 테스트를 작성해줘."
```

#### 단계 3: 우선순위 추가
```
AI에게 요청:
"TODO에 우선순위(high, medium, low)를 추가하고 싶어.
기존 함수를 수정해서 우선순위를 지원하도록 만들어줘."
```

#### 단계 4: 파일 저장 (도전!)
```
AI에게 요청:
"TODO 데이터를 JSON 파일로 저장하고 로드하는 기능을 추가해줘."
```
