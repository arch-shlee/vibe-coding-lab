## 📖 소개

**Vibe Coding Lab**은 초심자부터 전문가까지, 누구나 **단계별로 안전하고 효율적**으로 바이브코딩을 학습할 수 있는 체계적인 실습 환경입니다.

### 바이브코딩(Vibe Coding)이란?

바이브코딩은 개발자가 **'의도(Vibe)'**만 전달하면, AI가 코드를 생성하고 개발자는 방향을 조율하는 현대적 개발 방식입니다. 단순히 AI에게 코드를 맡기는 것이 아니라, **개발자가 선장이 되어 AI라는 똑똑한 부사수를 활용**하는 철학입니다.

### 왜 이 프로젝트가 필요한가?

- 🤔 **AI 코딩 도구는 많지만, 어떻게 제대로 활용할지 모르겠다**
- 🚨 **AI가 만든 코드가 불안하고, 테스트 없이 사용하기 두렵다**
- 🧩 **여러 AI 모델을 어떻게 조합해야 효율적인지 궁금하다**
- 📚 **TDD를 배우고 싶지만, AI 시대에 어떻게 적용할지 막연하다**

**Vibe Coding Lab**은 이 모든 질문에 대한 답을 **실습으로** 제공합니다.

## ✨ 특징

### 🎯 단계별 학습 로드맵
- **Level 1 (기초)**: 바이브코딩과 TDD 기본 개념
- **Level 2 (중급)**: 단일 AI 에이전트 최적 활용
- **Level 3 (고급)**: 멀티 에이전트 협업 시스템
- **Level 4 (실전)**: 프로덕션 레디 프로젝트

### 🛡️ 안전한 실습 환경
- 검증된 예제와 모범 답안
- 단계별 체크포인트
- TDD 기반 품질 보증

### 🤝 멀티 에이전트 시스템
- **Gemini 3 Pro**: 아키텍트 (설계, 기획)

- 역할 기반 협업 워크플로우

### 📚 풍부한 학습 자료
- 단계별 가이드 문서
- 실전 프로젝트 예제
- 커뮤니티 기반 지식 공유

## 🚀 빠른 시작

### 사전 준비

1. **Google Antigravity** 계정 (필수)
   - [안티그라비티 가입](https://antigravity.google.com) (가상 링크)
   - Gemini 3 Pro 및 Claude Sonnet 4.5 액세스

2. **기본 프로그래밍 지식**
   - JavaScript 또는 Python 기초
   - Git 기본 사용법

3. **개발 환경**
   - Node.js 18+ 또는 Python 3.9+
   - 코드 에디터 (VS Code 권장)

### 설치

```bash
# 1. 리포지토리 클론
git clone https://github.com/arch-shlee/vibe-coding-lab.git
cd vibe-coding-lab

# 2. Level 1 시작
cd level-1-basics

# 3. 첫 번째 모듈 진입
cd 01-hello-vibe

# 4. README.md 읽고 시작하기
cat README.md
```

### 첫 번째 실습 (5분)

```bash
# starter 폴더로 이동
cd starter

# 안티그라비티 열고 첫 바이브 전달하기
# "간단한 Hello World 프로그램을 만들어줘. 테스트도 함께."
```

## 📚 학습 경로

### 🟢 Level 1: 기초 (1-2주)

**목표**: 바이브코딩과 TDD의 기본 이해

```
level-1-basics/
├── 01-hello-vibe/          # 첫 바이브코딩 경험
├── 02-first-tdd/           # TDD 입문
├── 03-ai-assistant/        # AI 조수 활용
└── project/                # Todo List 앱
```

**완료 기준**:
- ✅ AI와 대화하며 코드 작성 가능
- ✅ Red-Green-Refactor 사이클 이해
- ✅ Todo List 앱 완성

### 🟡 Level 2: 중급 (2-3주)

**목표**: TDD 심화 및 단일 에이전트 마스터

```
level-2-intermediate/
├── 01-red-green-refactor/  # TDD 심화
├── 02-single-agent/        # 에이전트 최적화
├── 03-prompt-engineering/  # 프롬프트 엔지니어링
└── project/                # 쿠폰 시스템
```

**완료 기준**:
- ✅ 복잡한 비즈니스 로직을 TDD로 구현
- ✅ 효과적인 프롬프트 작성
- ✅ 쿠폰 시스템 완성

### 🟠 Level 3: 고급 (3-4주)

**목표**: 멀티 에이전트 시스템 구축

```
level-3-advanced/
├── 01-multi-agent-setup/   # 멀티 에이전트 설정
├── 02-architect-builder/   # 아키텍트-빌더 패턴
├── 03-workflow-optimization/ # 워크플로우 최적화
└── project/                # 주식 대시보드
```

**완료 기준**:
- ✅ 멀티 에이전트 시스템 독립 구성
- ✅ 에이전트별 역할 분담
- ✅ 주식 대시보드 완성

### 🔴 Level 4: 실전 (4-6주)

**목표**: 실무 적용 및 프로덕션 배포

```
level-4-mastery/
├── 01-real-world-project/  # 실전 프로젝트
├── 02-team-collaboration/  # 팀 협업
├── 03-production-ready/    # 프로덕션 준비
└── capstone-project/       # 최종 프로젝트
```

**완료 기준**:
- ✅ 실무 수준 프로젝트 완성
- ✅ CI/CD 파이프라인 구축
- ✅ Capstone 프로젝트 배포

## 🎓 학습 방법

### 1. 순차적 학습 (권장)
Level 1부터 순서대로 진행하세요. 각 레벨은 이전 레벨의 지식을 기반으로 합니다.

### 2. 체크포인트 활용
각 모듈에는 체크포인트가 있습니다. 이해도를 점검하고 다음으로 넘어가세요.

### 3. 프로젝트 중심 학습
각 레벨의 마지막에는 종합 프로젝트가 있습니다. 실전처럼 구현해보세요.

### 4. 커뮤니티 참여
- **GitHub Discussions**: 질문하고 답변하기
- **코드 리뷰**: 다른 학습자의 코드 리뷰
- **지식 공유**: 배운 내용 정리하여 공유
